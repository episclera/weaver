## Getting started

- Install: `npm i @episclera/weaver-icons`

```jsx
import React from 'react'
import { LoadingOutlined } from '@episclera/weaver-icons'

export default () => (
  <div>
    Loading... <LoadingOutlined />
  </div>
)
```
